Yutong Pan| 4/22/2017| All data sources from WHO

Global Information System on Alcohol and Health

This visualization describes the worldwide youth alcohol problem, it displays the percentage of current and former youth drinkers (15-19 years old) by country.

Tool Description

I choose tree map as my main visualization because the dataset including nearly 200 variables, which are county names. Therefore, displaying a bar chart or bubble chart would be difficult to keep track of all the countries. Also, the dataset include the region of places, so I choose to use colors for different regions in order to show differences between regions. I used buttons as my controlling element, so people can click buttons to see former female drinkers and current female drinkers. By clicking the buttons, the tree maps will show changes in different categories. Also, it is easy to tell the changes in each region as a whole, as well as compare between regions and countries.

For all controlling elements, it enables users to ask which region has the largest current/former male/female youth drinkers. It also allows users to ask is there a big difference between female and male drinkers as people can switch back and forth to see the changes. Users can also ask about which region changes the most from former drinkers to current drinkers. What I've found from this visualization is that Latin America and Europe have the largest former male drinkers, and there are not huge differences between former male drinkers and female drinkers. Europe also has the largest current male drinkers obviously, especially for female drinkers, there are more. Also, a lot of countries in each region have really similar percentages in general.
